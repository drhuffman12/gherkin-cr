# gherkin_cr

Gherkin parser/compiler for Crystal-lang, ported from [gherkin-ruby](https://github.com/cucumber/gherkin-ruby) to Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  gherkin_cr:
    github: drhuffman12/gherkin_cr
```

## Usage

```crystal
require "gherkin_cr"
```
